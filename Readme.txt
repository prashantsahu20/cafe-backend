


## Project Title - " Cafe Management Application "


The aim of this project is to develop a robust, user-friendly and efficient software solution 
that facilitates the management of Food Order by customers, managing the menu items and user data in a cafe.


## For this project I am using software technology like
* JDK 17 
* STS(Spring Tool Suite)
* My SQL Workbench 8.2 and
* PostMan
